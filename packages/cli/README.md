# @raazkhnl/rk-editor-cli

The official CLI tool for RK Word Editor to help you scaffold new projects quickly.

## Installation

```bash
npm install -g @raazkhnl/rk-editor-cli
```

## Usage

### Initialize a new project

```bash
rk-editor init my-awesome-editor
```

This will create a new directory with a pre-configured `package.json` and basic project structure.

## Commands

- `init [dir]`: Scaffolds a new project in the specified directory.
- `--version`: Shows the current version.
- `--help`: Shows all available commands.
